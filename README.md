## 🌿 About the Project

The **Smart Waste Management Portal** (*Eco Tycoon: Ultimate Trash Collector*) transforms the way people engage with waste management. Rather than a dry dashboard, this portal is a full-fledged **interactive eco-platform** — citizens can track garbage trucks live, report overflowing bins on a real map, take gamified quizzes, play sustainability arcade games, complete learning modules, and buy eco-friendly products, all in one seamless experience.

Built with a modern full-stack architecture — **React + Vite** on the frontend, **Node.js + Express + Prisma + MongoDB** on the backend — and deployed with **CI/CD via GitHub Actions**.

## 🗂️ Pages & Features

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [npm](https://www.npmjs.com/)
- A [MongoDB](https://www.mongodb.com/atlas) instance (local or Atlas)

### 1. Clone the Repository

```bash
git clone https://github.com/venkatasriharika-code/smart_waste_management_portal.git
cd smart_waste_management_portal
```

### 2. Configure Environment Variables

Create a `.env` file inside the `server/` directory:

```env
DATABASE_URL="your_mongodb_connection_string"
PORT=5000
```

### 3. Start the Backend

```bash
cd server
npm install
npm run dev
```

> ✅ Runs at `http://localhost:5000`

### 4. Start the Frontend

Open a new terminal:

```bash
cd client/wastmanagemet
npm install
npm run dev
```

> ✅ Runs at `http://localhost:5173`

> ⚠️ Always start the **backend first**, then the frontend.
